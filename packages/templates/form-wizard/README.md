# Form Wizard Template

## 목적

단계 표시, 입력 폼, 이전/다음 액션을 포함한 접수형 폼입니다.

## 사용하기 좋은 프로젝트 유형

견적 요청, 프로젝트 접수, 예약 신청, 온보딩 설문에 적합합니다.

## 필요한 import

```tsx
import { FormWizardPattern } from '@repo/ui';
```

## 커스터마이징 포인트

step 정의, input field, validation schema, 완료 CTA를 실제 요구사항에 맞게 연결합니다.

## 주의사항

실제 제출 전에는 접근 가능한 error message와 서버 검증을 반드시 추가하세요.
