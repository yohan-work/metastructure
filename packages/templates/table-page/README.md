# Table Page Template

## 목적

검색, 필터, 테이블, 페이지네이션이 있는 업무형 목록 페이지입니다.

## 사용하기 좋은 프로젝트 유형

고객 목록, 예약 목록, 주문 목록, 상담 티켓 목록에 적합합니다.

## 필요한 import

```tsx
import { TablePagePattern } from '@repo/ui';
```

## 커스터마이징 포인트

컬럼, row data, 상태 필터, pagination 동작을 실제 데이터 모델에 맞게 바꿉니다.

## 주의사항

대량 데이터는 클라이언트 전체 렌더링보다 서버 페이지네이션 또는 가상화를 검토하세요.
