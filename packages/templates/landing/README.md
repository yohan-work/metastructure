# Landing Template

## 목적

서비스 소개, 병원 홈페이지, 케이터링 소개, MVP 랜딩 페이지의 첫 버전을 빠르게 구성합니다.

## 사용하기 좋은 프로젝트 유형

예약, 문의, 상담, 견적 요청처럼 첫 화면 전환이 중요한 프로젝트에 적합합니다.

## 필요한 import

```tsx
import { LandingPattern } from '@repo/ui';
```

## 커스터마이징 포인트

hero 문구, feature 목록, use case copy, CTA 링크를 프로젝트에 맞게 교체합니다.

## 주의사항

앱 코드에서 Astryx를 직접 import하지 말고 `@repo/ui` 패턴 또는 컴포넌트를 확장하세요.
